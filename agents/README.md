# Agents

Custom Claude Code agent definitions — specialized subagents you spawn via the `Agent` tool.

## Layout per agent

```
agents/<agent_name>/
├── <agent_name>.md      Agent definition (frontmatter + system prompt)
└── README.md            What the agent is for, when to spawn it, example prompts
```

## Installation

Copy the agent definition file into the target project's `.claude/agents/` directory.
