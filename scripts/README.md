# Scripts

Helpers for working with the toolkit itself.

## Planned

- **`promote.sh <project-path> <component-name>`** — Diff a project's local copy of a component against the library version. Prints the diff so you can decide what to lift back into the library.
- **`install.sh <project-path> <component-name>`** — Copy a component from the library into a project's `.claude/` directory.

These are stubs — build them when the manual workflow gets annoying enough to justify automation.
